仿微信项目 旨在提升自己的实战能力

技术栈

| 前端         | 后端                 | 数据库 | 中间件 |        
| ------------ | -------------------- | ------ | ------ |
| Vue          | spring-gateway       | mysql  | rocketMq|
| element-plus | spring-boot          | redis  | nacos |
|              | dubbo                | ipfs   |       |
|              | websocket            | mongodb|       |

前端启动 npm run dev
后端 先启动user模块 不然会报找不到dubbo服务错误
